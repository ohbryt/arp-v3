## 3. 서비스 상품

### 3.1 약물 파이프라인 분석 (80만원/건)

**대상:** 바이오텍 스타트업, VC, 투자자

**분석 내용:**
```
[목차]
1. Executive Summary
2. 치료 영역 개요 (Epidemiology, Unmet Need)

8. 결론 및 투자 의견
```

## 4. AI 기반调研 시스템

### 활용 도구

| 도구 | 용도 |
|------|------|
| **TinyFish API** | 웹 자동调研 (PubMed, 임상시험DB, 규제当局) |
| **MERFISH 분석** | 피부과학 데이터 기반 분석 |
| **AI Content Pipeline** | 보고서 작성 자동화 |
| **Claude Code** | 데이터 분석, 시각화 |

### 업무 흐름

```
1. 고객 요청 수령
   ↓
2. TinyFish로 웹调研
   (PubMed, ClinicalTrials.gov, FDA, 논문)
   ↓
3. AI 분석 + Claude Code로 데이터 처리
   ↓
4. 보고서 초안 작성
   ↓
5. 전문가 검토 (Dr. OCM)
   ↓
6. 납품
```

**예시:** NASH 치료제 분석의 경우
- TinyFish: FXR, PPAR, GLP-1, THR-β 관련 논문+시험 수집
- MERFISH: TGF-β 경로 데이터 분석
- 보고서: 80만원/건 (2-3일 소요)
