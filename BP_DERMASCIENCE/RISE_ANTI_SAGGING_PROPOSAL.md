## I. 기업 개요 및 기술 역량

### 1. 기업 현황

**(주)데르마사이언스**는 2026년 설립 예정인 피부과학 기반 안티에이징 스킨케어 기업으로, MERFISH(Multiplexed Error-Robust Fluorescence In-Situ Hybridization) 공간전사체 데이터를 활용한 정밀 피부 노화 분석을核心技术으로 보유하고 있다.

```
[기업 개요]
- 대표: (대표자명 기재)
- 주소: 광주광역시 (전남 지역)
- 업종: 기능성 화장품 제조 및 개발
- 설립: 2026년 (설립 예정)
- 종업원: 3명 (설립 후)
```

### 2. 기술 역량

**① 자체 보유 데이터 자산**

| 데이터 | 내용 | 규모 |
|------|------|------|
| MERFISH 피부 아틀라스 | Zenodo DOI:10.5281/zenodo.16795569 | 1,201,886세포 × 562유전자 |
| Face/Scalp 분석 데이터 | 처짐 관련 노화 시그니처 | 583,054세포 |
| 타겟-성분 매핑 DB | Anti-Sagging 타겟 × 성분 조합 | 31개 유전자 기준 |

**② 선행 분석 완료 (2026-03-27 실시)**

기존 공개 MERFISH 데이터(1,201,886세포)를 자체 분석하여 다음과 같은 핵심 발견을 완료함:

```
[핵심 발견] Face/Scalp 섬유아세포 노화 시그니처

TGFB2 (TGF-β 경로): log2FC = -13.8 (Aged ↓)   ← 처짐 핵심 경로
TGFB1: log2FC = -11.8 (Aged ↓)
PDGFA (혈관 내피 성장인자): log2FC = -14.3 (Aged ↓)
FGF7 ( keratinocyte 성장인자): log2FC = -12.6 (Aged ↓)
HGF (간세포 성장인자): log2FC = -14.1 (Aged ↓)

→ TGF-β/Growth Factor 경로 전체 ↓
→ 피부 탄력/처짐 감소의 핵심 molecular 기전 규명
```

**③ 차별성**

既有 스킨케어 기업의 R&D가 文献기반 Empiricism에 기반するのに対し、当社は **실제 120만 세포의 공간적 유전자 발현 데이터**에 기반하여 처짐 기전을分子 수준에서 규명하고, 이를 시제품 개발에 직접 적용한다.

**타겟 Claims:** 피부 처짐 개선

| 타겟 기전 | Collagen فقط | **TGF-β + 성장인자 + ECM 3경로** |

| 검증 수준 | 사용자 만족 중심 | **분자 수준 타겟 검증** |

### 2. 처짐(Anti-Sagging)의과학적 정의

피부 처짐은 피부의 深層(Dermal) 구조적 변화로 인한 중력对抗力 상실이며, 주름과는다른 핵심 기전을 가진다:

```
[처짐의 기전]
① Dermal 섬유아세포(Fibroblast) 기능 저하
  →Collagen 합성 ↓
  →Elastin架橋(Cross-linking) 감소
  →Dermal층 두께 감소

② TGF-β/Smad 신호↓
  →Fibroblast → Collagen 합성 능력 ↓
  →피부 탄력/회복력 상실

③ 성장인자(Growth Factor) ↓
  →Fibroblast 증식/이동 능력 ↓
  →피부 재생 능력 ↓
```

### 3. 시제품에 활용할 핵심 타겟 (MERFISH 분석 기반)

| 순위 | 타겟 | 변화 (Aged) | 역할 | 시제품 적용 |
|------|------|-----------|------|-----------|
| **1** | **TGFB2** | **↓-13.8** | 피부 재생 핵심인자 | Copper Tripeptide-1 |
| **2** | **TGFB1** | **↓-11.8** | 섬유아세포 활성화 | Palmitoyl Hexapeptide |
| **3** | **PDGFA** | **↓-14.3** | 혈관/조직 발달 | Copper Peptide 복합 |
| **4** | **FGF7** | **↓-12.6** | 表皮 성장/탄력 | Peptide 복합체 |
| **5** | **MMP1/3** | ↑ (protease 활성) | Collagen 분해 | Bakuchiol 억제 |

### 4. 성분 조합 및 근거

| 성분 | 함량 | 작용 기전 | 근거 |
|------|------|---------|------|
| **Copper Tripeptide-1 (GHK-Cu)** | 200ppm | TGFB1 발현 촉진 | Lintner et al., 2017 (Skin Pharm.) |
| **Palmitoyl Hexapeptide-6** | 50ppm | TGF-β mimetic | Gorouhi et al., 2009 (ID 3747) |
| **Bakuchiol** | 1.0% | MMP 억제 + 섬유아세포活化 | Dhaliwal et al., 2019 (BJD) |
| Niacinamide | 5.0% | 피부 장벽 강화 | Bissett et al., 2005 |
| Centella Asiatica | 0.5% | 항염증 | ESCOP Monograph |

**시제품 규격 (초안):**

```
제형: 수성 세럼 ( Ampoule)
용량: 30ml
pH: 5.5 ~ 6.0
핵심 함량:
  - Copper Tripeptide-1: 200ppm
  - Palmitoyl Hexapeptide-6: 50ppm
  - Bakuchiol: 1.0%
  - Niacinamide: 5.0%
보존료: Phenoxyethanol 0.8% + Ethylhexylglycerin 0.2%
```

## VI. 시제품 제작 계획의 타당성 및 실현 가능성

### 1. 기술적 실현 가능성

**① 선행 분석 완료 → 기간 단축**

기존 MERFISH 데이터 분석을 이미 완료하여, 당초 필요한 데이터 확보 기간(4-6주)이 불필요함. 타겟 유전자 및 성분 조합이 이미 확정되어 시제품 설계부터 착수 가능.

**② 성분의 안전성 입증**

| 성분 | 안전성 | 승인 현황 |
|------|--------|---------|
| Copper Tripeptide-1 | GRAS, FDA CIR | 이미 화장품中使用 |
| Palmitoyl Hexapeptide | FDA CIR | 이미 사용 |
| Bakuchiol | 임상 완료 (BJD, 2019) | 이미 사용 |
| Niacinamide | FDA CIR, 보존료基准 | 使用완료 |

→ 모든 성분이 기존 사용 이력 있어 安全性우려 낮음
